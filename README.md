# 囚人のジレンマ シミュレーター

協力と裏切りのジレンマを体験するインタラクティブ教材

🔗 **公開URL**: https://dilemma.meetupsensei.com/

## 概要

「協力」か「裏切り」かを選び合う囚人のジレンマを、様々なAI戦略と対戦しながら体験できるシミュレーターです。しっぺ返し・常に協力・常に裏切りなどの戦略を相手に選んで複数ラウンド対戦し、累積スコアの推移をグラフで確認できます。得点ルールのカスタマイズや考察メモ機能も搭載し、ゲーム理論の基礎を直感的に学べます。日本語/英語切り替え・ダークモード対応。

## 使い方

- 対戦相手の戦略（しっぺ返し・常に協力・常に裏切りなど）を選ぶ
- ラウンド数と得点ルールを設定して「ゲーム開始」をクリックする
- 各ラウンドで「協力」か「裏切り」かを選択する
- 対戦記録と累積スコアグラフで結果を振り返る
- 考察欄に気づきをメモして戦略を分析する
